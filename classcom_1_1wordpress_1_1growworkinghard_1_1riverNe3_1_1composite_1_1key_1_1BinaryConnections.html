Detailed Description

Connection class for Binary Tree

This class describes the connections of each node of a Binary Tree. It allows null value for both left child and right child. For this reason, the validateState() method checks only the ID and the PARENT keys. The validateInvariant() method checks the connections thinking about the possibility that children might be null as well.

This class is ThreadSafe because it is Immutable as defined in [3]
