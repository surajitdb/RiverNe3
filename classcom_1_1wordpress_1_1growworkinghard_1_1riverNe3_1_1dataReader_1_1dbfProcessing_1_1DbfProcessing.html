Detailed Description

abstract class DbfProcessing

This class is useful as interface, in order to guide the developer who wants to write a class which parses a .dbf file of a .shp.

The implemented methods are synchronized because the reading of each file can be done just with one thread, but files can be read simultaneously by threads.
