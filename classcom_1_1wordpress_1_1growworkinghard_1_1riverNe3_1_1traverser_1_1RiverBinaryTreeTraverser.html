Detailed Description

How retrieving right and left children from the binary tree.

This class implements how retrieving right and left children from the HashMap containing the binary tree. The implementation of the traverser are in the BinaryTreeTraverser class, from the GUAVA package

This class is ThreadSafe because it is Immutable.
