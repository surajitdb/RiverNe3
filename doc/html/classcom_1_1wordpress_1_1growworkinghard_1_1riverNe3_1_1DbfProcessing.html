Detailed Description

class DbfProcessing

This class parses the dbf of a shapefile and return a List filled with the desiderd Geometry objects.

This class is ThreadSafe because it has been designed following the Stack-confinement principle [1]

DbfProcessing dbfProc = new DbfProcessing();
List<Geometry> test = dbfProc.get(filePath, geomType, colNames);

Getter method which return the list of geometric features.

This is the only one public method. It returns a synchronizedList of objects of type Geometry.

Parameters
filePathThe complete input path of the .dbf file
typeThe type of geometry of the shapefile
colNamesAn array of strings of the column names to search. It has to be structured in the following way
  1. Line:
    • Column name of the Pfafstetter numbering;
    • Column name of the X coordinate of the starting point;
    • Column name of the Y coordinate of the starting point;
    • Column name of the X coordinate of the ending point;
    • Column name of the Y coordinate fo the ending point.
Returns
A Collections.synchronizedList of the filled list
